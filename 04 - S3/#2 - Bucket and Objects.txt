Bucket

    - Bucket Names must be unique across ALL of AWS.
    - Bucket Names must be 3 to 63 characters and contains only lowcases letters
    - Bucket Names must not be formatted as an IP Address.

    - Properties Info:
        - Bucket Level:
            - General Info
            - Permissions
            - Static Web Hosting
            - Logging
            - Events
            - Versioning
            - Lifecycle
            - Cross-Regions
            - Tags
            - Requesters Pays
            - Transfer acceleration

        - Folders
            - General Info
            - Details

        - Objects
            - General Info
            - Details
            - Permissions
            - MetaData
